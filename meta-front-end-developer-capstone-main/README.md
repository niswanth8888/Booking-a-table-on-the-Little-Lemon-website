# Little Lemon Web Application

This is a React-based web application created for the **Little Lemon Restaurant** project.  
The application allows users to explore the restaurant website and book a table using a functional reservation form.

## Project Overview

The main goal of this project is to build a responsive and user-friendly restaurant web application using React.  
The project includes a table booking form with proper form handling, validation, and accessibility features.

## Features

- Responsive restaurant website layout
- Table booking form
- Form validation for required fields
- Date and time selection
- Guest count selection
- Occasion selection
- User-friendly error messages
- Accessible form labels and inputs
- Clean React component structure
- Mobile-friendly design

## Technologies Used

- React
- JavaScript
- HTML5
- CSS3
- Git
- GitHub

## Booking Form

The booking form allows users to enter details such as:

- Name
- Email or contact details
- Booking date
- Booking time
- Number of guests
- Occasion

The form validates the user input and shows proper error messages when required fields are missing or invalid.

## Accessibility

This project follows basic accessibility practices, including:

- Semantic HTML elements
- Proper labels for form inputs
- Clear button text
- Readable layout
- User-friendly validation messages
